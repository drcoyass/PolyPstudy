# Poly-Pstudy (ポリリン酸研究会データベース) アーキテクチャ設計案

## 💡 プロジェクト概要
ポリリン酸（Polyphosphate）に関する最新の論文情報の収集、AIによる要約、および毎月の動画・スライド資料作成のプロセスを完全に自動化し、会員に向けて継続的に価値を提供する「最強のデータベースHP」を構築します。

---

## 🏗️ システム機能と構成

### 1. 情報収集・スライド自動作成パイプライン (`monthly_slide_generator.py`)
- **技術スタック**: Python (OpenAI API, python-pptx, PubMed E-utilities)
- **全体の流れ（論文自動収集・要約・スライド作成）**:
  1. **論文自動収集**: PubMed APIを利用して、「Polyphosphate」の最新論文 (タイトル、アブストラクト、著者など) を自動検索＆取得します。
  2. **自動要約**: OpenAI API (GPT-4o等) にアブストラクトを送信し、専門的なリサーチャーの視点でスライドに載せやすい簡潔な箇条書き形式（3〜4つ）に自動要約させます。
  3. **自動スライド作成**: `python-pptx`ライブラリを使用し、各論文の情報を綺麗にフォーマットしたPowerPoint (.pptx) ファイルを自動生成します。「ポリリン酸 最新研究レポート」としてそのまま動画化やPDF化に利用できます。

### 2. フロントエンド（最強のデータベースHP）
会員が見やすく、プレミアム感のあるモダンなWebサイト。
- **実装ファイル**: `index.html`, `style.css`, `app.js`
- **主な特徴と機能**:
  - **洗練されたデザイン**: ダークモードとグラスモーフィズムデザインを取り入れ、最先端の研究会にふさわしいPremiumな体験を提供。
  - **論文・資料ダッシュボード**: 自動生成されたスライド資料（PDFなどに変換して掲載）や最新論文の一覧を閲覧・検索可能。
  - **NotebookLM連携**: ワンクリックで論文リファレンスをコピーできるボタンを実装し、Google NotebookLMとスムーズに連携して自己学習やディスカッションに活用できます。

## 🚀 運用ダッシュボードと今後の展開

* **必要な準備・セットアップ (動作環境)**:
  1. `pip install -r requirements.txt` を実行して必要なライブラリ (`openai`, `python-pptx`) をインストール。
  2. `OPENAI_API_KEY` を環境変数として設定。
  3. 月に1回（あるいは定期的に）、`python3 monthly_slide_generator.py` を実行するだけで「PolyP_Monthly_Report_YYYYMM.pptx」が生成されます。

* **Next Steps (任意)**:
